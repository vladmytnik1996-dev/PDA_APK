# Состав GitHub-пакета 1.0.6

## Для пользователя

- `apk/stalker-pda-v1.0.6-release.apk` — подписанный полноэкранный Android APK.
- `README.md` — подробное руководство пользователя со скриншотами.
- `docs/USER_GUIDE.html` — та же инструкция в формате, который можно открыть браузером.
- `docs/USER_GUIDE.md` — Markdown-версия инструкции.
- `stalker-pda-web-v1.0.6.html` — автономный однофайловый сайт.
- `web/` — вариант для запуска через локальный сервер на Android; открывать именно `http://localhost:8080/`.

## Исходный код

- `source/web-app/` — исходный однофайловый HTML.
- `source/android-project/` — Android WebView-проект, Smali-исходники оболочки, инструменты сборки и тесты.

## Контроль качества

- `CHANGELOG.md` — изменения версии.
- `TEST_REPORT.md` — подробный отчёт по выполненным тестам и ограничениям.
- `tests/` — компактные результаты автоматических проверок.
- `SHA256SUMS.txt` — контрольные суммы файлов пакета.

## Конфиденциальность

В документации используется только условная демонстрационная карта. Реальная игровая карта и приватный release-keystore в пакет не входят.
