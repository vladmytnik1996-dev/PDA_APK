# КПК Зоны — Android fixed v6 fullscreen

Версия `1.0.5` (`versionCode 10005`).

## Главное изменение

Приложение запускается в immersive sticky fullscreen:

- скрыты часы, сеть, уведомления и остальные значки верхней системной строки;
- скрыта нижняя системная панель навигации;
- после возврата из разрешений, выбора файлов, камеры и сворачивания полноэкранный режим восстанавливается;
- интерфейс учитывает безопасные области экранов с вырезами и жестовой навигацией.

## Установка

Release v6 имеет тот же package name и release-сертификат, что v5:

- package: `by.smorhon.stalkerpda.v3`;
- versionName: `1.0.5`;
- versionCode: `10005`.

Устанавливайте APK поверх v5, не удаляя приложение. Это сохраняет локальную карту, метки и заметки.

## Сборка

Debug:

```bash
./build.sh debug
```

Release требует прежние файлы подписи в `signing/`:

```text
signing/stalker-pda-release.jks
signing/release-password.txt
```

После этого:

```bash
./build.sh release
```

Полная проверка:

```bash
./verify.sh
```

## Системные жесты

Android может временно показать системные панели после свайпа от верхнего или нижнего края. После завершения системного жеста приложение снова возвращается в immersive-режим.
